# Skills

This directory contains repo-local skills for agent workflows in BrowserTrace Harness.

## Available skills

### `browsertrace-v2-fullstack`

Path:

- [skills/browsertrace-v2-fullstack/SKILL.md](/Users/lotosli/Documents/BrowserTrace Harness/skills/browsertrace-v2-fullstack/SKILL.md)

Use this skill when an agent is doing full-stack development or debugging with BrowserTrace V2 and needs to:

- run one-shot scenarios with `browsertrace run`
- drive persistent debugging sessions with `run-session`
- use `--through-step` to advance a scenario incrementally
- use `run-session judge` for session-level scenario truth
- interpret `verdict`, `diagnosis`, and `artifacts`
